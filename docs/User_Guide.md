# CRM System user guide

Sign in with your ABC Company role. Sales users can open Contacts to search names/email and filter tags, create or edit contact records, and move Deals through the pipeline. Support users can open Support, sort tickets by priority and age, read the customer context permitted by their role, assign tickets, and set lifecycle states. Analysts can view aggregated Reports without editing operational records. Admins can assign roles, import validated CSV contact records, merge approved duplicates, and manage controlled notification templates.

Do not copy customer data to unmanaged spreadsheets. Use CSV import only for the approved legacy migration format. When validation rejects a row, correct that row and upload again; valid records are not silently merged.

For a production incident, record the request id and CRM ticket number, avoid pasting credentials or personal data into chat, and notify Priya Sharma and Vikram Singh. SendGrid delivery and AWS deployment operations are described in the deployment guide.
